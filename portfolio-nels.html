New England Language School Website

Static website for the New England Language School in Quincy, MA
  • project name :NELS Static Website
  • client :NE Language School
  • launch date : may, 2019
see live site
This is a professional project for the New Englang Language School in Quincy, MA. It is a static website created using HTML, CSS and Javascript. This site was created 100% by hand from the bottom up; no template, plugins, packages or any add ons used.
This project involved working closely with the business owner to understand the functional needs of the website as well as stylistic choices to convey the image and branding of the business desired by the client. The project was executed by utilizing good communication skills with the client as well as using my technical knowledge to create the website and consult the client.
