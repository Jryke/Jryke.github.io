Shooting Range Game

Game created using HTML, CSS & JavaScript
  • project name : Shooting Range Game
  • client : Personal Project
  • launch date : September, 2019
  • GitHub : Shooting Range Repo
see live site
This a simple game that simulates a shooting range. The targets are archery style and give points depending on how close to the middle they are hit (bullseye = 10 points, outer ring = 1 point). The mouse is used to shoot and each shot adds to the shot count as well as the point count (if a target is hit). The targets move up and down and in the process are hidden for random short periods of time.
The layout including the targets is made with HTML and CSS. Vanilla JavaScript is used to make the targets move at random intervals, make interface interactive with mouse click "shots", as well as calculate and render the points.
