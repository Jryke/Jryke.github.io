CourseFill

Web app to facilitate registration of students for educational courses
  • project name : CourseFill
  • client : Enterprise Software
  • completion date : January, 2020
see development site
CourseFill is an online app for use by educational institutions. The main purpose is for students to view, register and make payments for courses. In addition to viewing course information, students will be able to view information about teachers. Teachers can create profiles as well as keep track of their courses, schedule and students.
CourseFill uses React.js on the front end. The server side is set up using Node.js and Express. The application data is stored using MongoDB. The shopping cart and payment for courses is set up using Stripe.
