Recipe App

Online tool to create and reference recipes
  • project name :Recipe App
  • client :Personal project
  • launch date : march, 2019
see live site
This is a simple app to enter recipes and reference them. There are two views, the "main" page (list of recipes) and the "edit" page (to view and edit recipe details). The user can select the ingredients that they have on the edit page. On the main page, each recipe has an ingredient status of "all", "some" or "none" which can also be used to filter only recipes that the user has all ingredients.
It is created with HTML and made responsive with CSS. This project was based around using JavaScript for various capabilites including filtering data, saving in local storage and most importantly DOM manipulation.
