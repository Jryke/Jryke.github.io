Whitneys of Harvard Square Website

Website for the Whitneys Bar in Harvard Square
  • project name :Whitneys Website
  • client :Whitneys of Harvard Sq
  • launch date : march, 2020
see live site

Technologies: AMP stack (Apache, MySQL, PHP), HTML, CSS & JavaScript

Libraries: Bootstrap 4

This is a professional project for Whitneys of Harvard Square. The AMP stack was utilized to create a custom CMS allowing bar tenders to easily update the list of beers "Currently on tap". The CMS is intentionally very simple with an intuitive user interface so that the bar tenders can focus on their job of serving and not be distracted by learning tech.
The intention of this project was to have a clean and simple (single page) website. The client's basic needs were to have an internet presence for the bar (previously the bar did not have a website), to give the website's visitors access to the bar's contact information/location and to keep them visitors informed with up-to-date beers on tap selection.
